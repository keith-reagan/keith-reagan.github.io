Basic Shell Script Maker


This is a short script I had created to make empty shell scripts for my Intro to Linux Class. I might not sound like much but it did save some time with assignments

					#!/bin/bash
if [ $# -lt 1 ]
then
	echo "Enter name for your script"
	read filename
else
	filename=$1
fi
case $filename in 
	*.sh)
		echo "Creating: $filename"
		touch $filename
		chmod 755 $filename;;
	*)
		echo "Creating: $filename".sh""
		touch $filename".sh"
		chmod 755 $filename".sh"
esac
echo "File created"
