Choose your own adventure story script


This is a short script I had created for class, it's a "Choose your own adventure" mini game. It might not be too long but hopefully you'll at least get a chuckle out of it.

						#!/bin/bash
echo "You've come to a fork in the road. Choose L or R" #prints the first line of the story
while read fork #entering the loop for whatever value is in fork
do #(to the Rammstein tune) do....do while.....do while loop
case $fork in #looks at the values in fork
	[lL]) #if it contains l or L 
	echo "You walk off to the left, you see a nice lake. Care to go fishing? Y or N" #it'll print this 
	while read fishing #entering the loop for the fishing value
	do #this!
	case $fishing in #looks at the value in fishing
		[yY]) #if it contains y or Y
		echo "You decide to go fishing, several hours pass and you forget to do your CIS117 lab!" #it'll print this
		break #leave this loop
		;; #end of statement
		[nN]) #if it contains n or N
		echo "You decide now isn't a good time to fish. Besides, you have your CIS117 lab to do!" #it'll print this
		break #leave this loop
		;; #end of statement
		*) #if it contains anything thats not nNyY
		echo "You need to enter Y or N" #it'll print this
	esac #we're gotta get outta this case
	done #with the loop
	break #break outa it
	;; #end of it
	[rR]) #just like the lL
	echo "You keep walking and see some thick brush, do you keep walking? Y or N" #it'll print this
	while read walking #entering the loop for walking value
	do #starting of the do part of the do while loop
	case $walking in #looks at the value given to walking
		[yY])	#if its y or Y
		echo "You keep walking not being able to see where you're going. You accidently fall off a cliff" #it'll print this
		break #ing up is hard to do. but you need to leave this loop now
		;; #end of statement	
		[nN]) #if its a n or N
		echo "You're right, you should probably head back and do your lab" #it'll print this
		break #up this code with some humor, and leave the loop
		;; #ending 
		*) #if its anything but yYnN...
		echo "You need to enter Y or N" #print this
	esac #case backwards, it also leaves the case
	done #with the loop		
	break	#down, shake down, busted out of the loop
	;;	#I want my ends
	*) #if its anything but lLrR...then....you guessed it....
	echo "You need to enter L or R" #print this, with extreme prejudice
esac #getting out of this case is the last thing this loop will ever do

done #yey!
echo "end of story" #prints that we're done here with the story. a real emotional rollercoaster. I hope this was creative enough!
